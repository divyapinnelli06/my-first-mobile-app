# IOT EggTray  Monitoring & Bulb Control App
 1.This is a mobile application developed using Kodular to an  egg tray status and remotely control a bulb using Fire      Realtime Database.
  2.The app reads live sensor values from hardware and updates the UI instantly.

 # Features 
   1.Egg tray Monitoring
     displays whether the egg tray is occupied or empty based on sensor data.
   2.Bulb ON/OFF Control
    Two buttons (ON/OFF) to control a bulb connected to an an IOT device.
   3.Realtime Firebase Sync
    The app communicates directily with Firebase Realtime for instant updates.
   4.Built entirely using Kodular 
     No coding required-drag and-drop logic

 # Technologies used
    1.Kodular Creator- for building the Android app.
    2.Firebase realtime Data base -for storing sensor values and light status.
    3.IoT Hardware-Raspberrypi
    4.Internet connectivity-for live communication

  # How it works
    1.IOT device reads egg tray sensor value.
    2.Sensor value is pushed to Firebase under sensorData1/eggtray.
    3.Kodular app reads this value and dispays occupied or Empty.
    4.When user presses the ON or OFF,the app updates lightstatus.
    5.IOT device reads lightStatus and turns the bulb ON/OFF.
